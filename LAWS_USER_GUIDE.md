# Laws Page - User Guide

## Overview
The Laws page provides easy access to Pakistani women's rights legislation with full text, searchable content, and downloadable PDFs.

## Features

### 1. Browse All Laws
**Location**: `/laws`

**What You'll See**:
- Grid of law book cards
- Each card shows:
  - Law title
  - Brief description
  - Category badge (e.g., "Family Law", "Workplace Rights")
  - Number of sections
  - "Read More" button

**Available Actions**:
- **Search**: Type in the search box to filter laws by title or description
- **Filter by Category**: Use the dropdown to show only laws in a specific category
- **Click "Read More"**: Opens the full law with all sections

### 2. View Law Details
**Location**: `/laws/:id` (e.g., `/laws/muslim-family-laws-1961`)

**What You'll See**:
- Law title and metadata
- Category and section count
- Two action buttons:
  - "Discuss this law in Chat" - Opens AI chat
  - "Download PDF" - Downloads the official PDF (if available)
- Expandable sections in an accordion

**Reading Sections**:
1. Each section is collapsed by default
2. Click on any section to expand it
3. You'll see:
   - Reference number (e.g., "Section 5", "WR-003")
   - Category label
   - Full legal text
   - Related theme tags (e.g., "marriage", "harassment", "justice")
4. Click again to collapse

### 3. Download PDFs
**Available for**:
- Pakistan Penal Code
- KP Enforcement of Women's Property Rights Act, 2019
- Muslim Family Laws Ordinance, 1961
- Prevention of Electronic Crime Act, 2016
- Protection Against Harassment at Workplace Act, 2010
- Punjab Protection of Women Against Violence Act, 2016

**How to Download**:
1. Open any law detail page
2. Click "Download PDF" button (if available)
3. PDF downloads to your device

### 4. Discuss in Chat
**How it Works**:
1. Click "Discuss this law in Chat" button
2. Opens the AI chat interface
3. Ask questions about the law
4. Get explanations in simple terms

## Example Use Cases

### Finding a Specific Law
1. Go to `/laws`
2. Type "harassment" in search box
3. See "Protection Against Harassment at Workplace Act, 2010"
4. Click "Read More"

### Reading About Marriage Rights
1. Go to `/laws`
2. Select "Family Law" from category filter
3. Click on "Muslim Family Laws Ordinance, 1961"
4. Expand "Section 5" to read about marriage registration
5. Expand "Section 6" to read about polygamy rules

### Downloading for Offline Reading
1. Open any law detail page
2. Click "Download PDF"
3. Read offline on your device

### Getting Help Understanding
1. Open any law detail page
2. Click "Discuss this law in Chat"
3. Ask: "Can you explain Section 6 in simple terms?"
4. Get AI-powered explanation

## Tips

- **Use Search**: Quickly find laws by typing keywords
- **Filter by Category**: Narrow down to specific types of laws
- **Expand Multiple Sections**: You can have multiple sections open at once
- **Download PDFs**: Keep copies for offline reference
- **Ask AI**: Use the chat feature to understand complex legal language

## Mobile Experience

The Laws page is fully responsive:
- **Mobile**: Single column layout, easy scrolling
- **Tablet**: 2-column grid
- **Desktop**: 3-column grid

All features work seamlessly on any device.

## Accessibility

- **Keyboard Navigation**: Use Tab to navigate, Enter to open sections
- **Screen Readers**: Full support for visually impaired users
- **High Contrast**: Works with system accessibility settings
- **Readable Text**: Clear fonts and proper spacing
